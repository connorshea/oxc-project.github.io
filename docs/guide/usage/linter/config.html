Oxlint Configuration File â€‹

This configuration is aligned with ESLint v8's configuration schema (eslintrc.json).

Usage: oxlint -c oxlintrc.json

NOTE

Only the .json format is supported.

Example

json
// oxlintrc.json
{
  // Comments are supported.
  "env": {
    "browser": true
  },
  "globals": {
    "foo": "readonly"
  },
  "settings": {},
  "rules": {
    "eqeqeq": "warn"
  }
}
